Это набор данных для сбора информации с сайта owen.ru в формате JSON.

Цель: автоматизированный сбор данных о модификациях товаров (наименования, артикулы, параметры) 
для последующего использования при формировании торговых предложений и интеграции на сайт.

### Структура проекта:
- owen_spider.py — паук на Scrapy + Selenium для парсинга карточек товаров.
- result.json — файл с результатом парсинга в формате JSON.
- Readme.txt — текущий файл с описанием.
- Запрос для ИИ.txt — техническое задание и описание задачи.

### Требования:
- Python 3.8+
- Библиотеки: scrapy, selenium, webdriver-manager

### Как запустить:
1. Установите зависимости:
   pip install scrapy selenium webdriver-manager
2. Перейдите в папку со скриптом:
   cd /путь/к/папке/
3. Запустите паука:
   scrapy crawl owen_mods_spider

Результат будет сохранён в файле `result.json`.

Для изменения логики парсинга отредактируйте `owen_spider.py`.
